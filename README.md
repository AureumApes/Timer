# Timer
##### By AureumApes

## How to use
### Windows, MacOS:
Download Code
<br>
Run `npm i` to install dependencies
<br>
Run `electron index.js` to start the app
### Linux
1. Download [this](https://github.com/AureumApes/Timer/releases/download/v.1.0.1/Timer.tar.xz)
2. Unpack it
3. Run the Timer Binary-File
