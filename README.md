# Timer
##### By AureumApes

## How to use
### Windows, MacOS:
1. Download Code
2. Run `npm i` to install dependencies
3. Run `electron index.js` to start the app
### Linux
1. Download [this](https://github.com/AureumApes/Timer/releases/download/1.1.0/Timer.tar.xz)
2. Unpack it
3. Run the Timer Binary-File
